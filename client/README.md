# Factorio Control Center Client

This directory contains the React/Vite single-page application for Factorio
Control Center.

Most contributors should start from the repository root. The root `README.md`
contains the full setup, configuration, and release instructions.

## Development

Install client dependencies:

```bash
npm install
```

Run the Vite development server:

```bash
npm run dev
```

The development UI runs at <http://127.0.0.1:5173/login> and expects the NestJS
API to be running from the repository root.

## Commands

```bash
npm run dev       # Start Vite
npm run build     # Type-check and build the client
npm run lint      # Run client ESLint
npm run preview   # Preview the built client
```
